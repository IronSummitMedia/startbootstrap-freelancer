Coin Compare

Coin Compare is a project I worked on with three other developers. Coin Compare is an application that was created for people who have little to no knowledge on crypto currency exchanging but would like to get into it. This project was created in Visual Studio Code in HTML, CSS, and Javascript as well as AJAX to call API's from Coinlib for back end storage and JSON was also used. Firebase was also used in the creation of this application to store user input data on the contact page as well as the blog page. This project was created on GitHub in a repository and forked between three other people. My Role in the creation of this project was the UI/UX graphics and visuals that you see. I created the graphics using a combination of Illustrator, Photoshop, After Effects, and Premiere. Click the image to view the application.

Giftastic Animals

Giftastic Animals is an application I created using HTML, CSS, Javascript, and AJAX to call an API key that generates Gif's of animals at random that are displayed and stored. Once the Gif's are displayed they are static until clicked, when clicked they play on a loop until clicked again and they stop. The user can also type in an animal in the search bar and when submit is clicked that animal will appear as a clickable button to reveal new GIF's. All of the UX / UI design was also done by myself and this project was created and launched in a GitHub repository. Click the image to view the application.
